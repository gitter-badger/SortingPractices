# SortingPractices
practicing of all the common sorting algorithms.

## how to test
	$ git clone https://github.com/RedBug312/SortingPractices.git
	$ cd ./SortingPractices
	$ make compilation
	$ make execution

## finished

* SelectionSort
* InsertionSort
* BubbleSort
* QuickSort
* RadixSort
